#LiteInterp R1
Ele tem 13 comandos

sum - Soma
min - Subtração
mult - multiplicação
div - divisão
say - equivale ao "print"
def - Vem de "define", serve para criar "funções"
!enddef - como não tenho editor de texto, será tudo digitado direto na linha de comando, por isso terei que colocar isso para finalizar uma função
!endif - Termina um if
if - Se
else - Senão
ask - Equivale ao "scanf", pergunta alguma coisa e armazena em algum lugar
put - armazena alguma coisa na variável que o usuário criar com "var"
var - cria uma variável

Operadores
"->" - "Jogue para", ele coloca algum valor de ask em uma variável

exemplo:

{
var apple = 1;
var orange = 2;
var result;
def whatFruit
(
  say"Escolha entre 1 e 2";
  ask -> result;
  if result ==> apple
  (
    say="Is Apple!";
  )
  else
  (
    say="is Orange!";
  )
  !endif
)
!enddef
}

Como ficaria quando eu fosse usar isso no shell?

shell@: liteinterp {var apple = 1;var orange = 2;var result;def whatFruit(say="Escolha entre 1 e 2";ask -> result;if result ==> apple(say="Is Apple!";)else(say="is Orange!";)!endif)!enddef}
